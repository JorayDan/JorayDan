FatRat生成APK后门

2018-08-29The total word count - 123Estimated time of reading - 1 mins

root@kali:~# git clone https://github.com/Screetsec/TheFatRat.git
root@kali:~# apt-get update
root@kali:~# apt-get install mingw-w64 backdoor-factory

apt-get update报错解决办法：

修改源：
vi /etc/apt/sources.list

阿里云
deb http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
deb-src http://mirrors.aliyun.com/kali kali-rolling main non-free contrib

root@kali:~# wget -q -O - https://archive.kali.org/archive-key.asc | apt-key add
OK

chmod +x fatrat
chmod +x powerfull.sh
./setupsh
./fatrat

生成apk选择1，再选择5就可以生成一个后门apk
再开启msf监听就可以
